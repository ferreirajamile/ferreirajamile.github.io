o que não é o amor

Jamile Ferreira, 7.jul.2013
 
Fonte A- | A | A+
 

O que é o amor? Não sei.

Só sei que
O amor não é frustração; o amor é pleno!
O amor não é tormento; o amor sonha solto…
O amor não é inércia; o amor dança!
O amor não é ofensa; o amor admira!

O amor não é condição; o amor doa!
O amor não é negócio; o amor é de graça, toma!
O amor não é prazo; o amor não sabe nunca a hora de parar!
O amor não é peso; o amor levita!!!

O amor não é desconfiança; o amor é crédulo!
O amor não é acomodação; o amor se move…
O amor não é carência; o amor está completo!!!
O amor não é falta; o amor está satisfeito, obrigada!

O amor não é cobrança; o amor se interessa.
O amor não é julgamento; o amor sabe perdoar.
O amor não é ausência; o amor está sempre por perto.
O amor não é crítica; o amor sabe que não é perfeito.

O amor não é fantasia; o amor é um banho de realidade.
O amor não é silêncio; o amor é um barulhão cá dentro!
O amor não é imposição; o amor dá opções.
O amor não é omissão; o amor tá em todas!

O amor não é acusação; o amor dá o benefício da dúvida.
O amor não é tara; o amor sabe ter tudo sem desespero.
O amor não é divulgação; o amor basta.
O amor não é exposição; o amor é confidência.

O amor não é mentira; ele cresce na verdade.
O amor não é medo; o amor enfrenta.
O amor não é controle; o amor dá liberdade.
O amor não é delação; o amor é sempre cúmplice.

O amor não é chantagem; porque o amor não tem preço!
O amor não é manipulação; o amor tem muita personalidade!
O amor não é disputa; o amor é a história do outro.
O amor não é solidão; o amor sabe fazer companhia.

O amor não é prisão; o amor voa…
O amor não é vício; o amor abre…
O amor não é platéia; o amor sou eu e você;
O amor não é dependência; o amor quer, se você quiser!

O amor não é angústia; o amor dorme tranquilo!
O amor não é guerra; o amor almoça junto!
O amor não é tristeza; o amor dá festas todo dia!
O amor não é mudança; o amor é – onde for pra ser!

O amor não é histeria; o amor é um sorriso.
O amor não é prova; o amor sabe!
O amor não é cinismo; o amor não sabe o que é isso!
O amor não precisa de conceituação, porque o amor se sente.
